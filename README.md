## Professional Website (Next.js)

Stunning “enterprise meets edgy” personal website for **Kaká Cordovil**.

Includes:
- About me
- Career journey timeline
- Skills + education
- Portfolio section (placeholders for future projects)
- Contact links

## Getting Started

From this folder, install and run:

```bash
npm install
npm run dev
```

Open the printed `localhost` URL in your terminal (it’s usually `http://localhost:3000`, but if that port is taken Next.js will pick the next available port).

## Editing content

Profile content lives in:
- `src/lib/profile.ts`

Main page layout:
- `src/app/page.tsx`

## Build

```bash
npm run build
npm start
```
